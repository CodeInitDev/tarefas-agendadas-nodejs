## 🛠 **Stack de Desenvolvimento**

- **Backend**: NodeJS

## 📖 **Instruções Detalhadas**

Para começar, certifique-se de ter todas as dependências instaladas e siga o passo a passo cuidadosamente.

### Instale

NodeJS - v20

1. Instale a dependencias
```sh
yarn install
```

2. Rode o arquivo
```sh
node src/cron.js
```
